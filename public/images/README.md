# Images Folder

Place your images here:

## Required Images:

1. **hero-children.jpg** - Hero section image showing children learning together
   - Recommended size: 1200x800px or larger
   - Format: JPG or PNG
   - This is the main hero image displayed on the home page

2. **teacher-photo.jpg** - Teacher profile photo
   - Recommended size: 400x500px
   - Format: JPG or PNG

3. **location-map.png** - Map showing location
   - Recommended size: 400x300px
   - Format: PNG or JPG

## How to Add Images:

1. Copy your image files to this folder: `tuition-website/public/images/`
2. Name them exactly as listed above, or update the component files with your custom names
3. The images will be automatically available at `/images/your-image-name.jpg`
